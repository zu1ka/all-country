# Where in the World?

A responsive web application that displays information about countries from around the world with search and filter capabilities.

## Features

- **Country Display**: View all countries with their flags, population, region, and capital
- **Search Functionality**: Search for specific countries by name
- **Region Filter**: Filter countries by continent (Africa, Americas, Asia, Europe, Oceania)
- **Dark Mode**: Toggle between light and dark themes with persistent preference
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Country Selection**: Click on any country card to highlight it

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript (ES6+)
- [REST Countries API](https://restcountries.com/)

## Getting Started

1. Clone or download the repository
2. Ensure you have the following files:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Open `index.html` in your web browser

No build process or dependencies required!

## Usage

- **Search**: Type a country name in the search box and click "Search" or press Enter
- **Filter**: Select a region from the dropdown to view countries from that continent
- **Combine Filters**: Use search and region filter together for refined results
- **Dark Mode**: Click the "Dark Mode" button to toggle themes (preference is saved locally)
- **Select Country**: Click on any country card to highlight it

## API

This project uses the [REST Countries API v3.1](https://restcountries.com/v3.1/all) to fetch country data including:

- Country name
- Capital city
- Population
- Region
- Flag image

## Browser Compatibility

Works on all modern browsers that support:

- ES6+ JavaScript features
- CSS Grid
- CSS Custom Properties (variables)
- Fetch API
